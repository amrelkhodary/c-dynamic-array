Detailed Description

Type handling system for generic type operations in cdyar.

This file defines the type handling mechanism that allows cdyar to work with arbitrary data types through void pointers and generic operations.

cdyar_typehandler

Function pointer type for handling generic type operations.

Type handler functions are responsible for copying data between two void pointers based on the specified direction and size. This allows the dynamic array to work with any data type.

Parameters
left_voidptrPointer to the left operand
right_voidptrPointer to the right operand
directionDirection of assignment (left-to-right or right-to-left)
sizeSize in bytes of the data type being handled
codePointer to return code for error reporting

Generic type handler implementation using memcpy.

This is the default type handler that works with any data type by performing byte-wise copying using memcpy. It replaces the previous macro-based approach with type-specific handlers.

Parameters
left_ptrPointer to the left operand
right_ptrPointer to the right operand
directionDirection of the assignment operation
sizeSize in bytes of the data to copy
codePointer to return code (set to CDYAR_SUCCESSFUL on success, CDYAR_INVALID_INPUT if pointers are NULL or direction is invalid)
