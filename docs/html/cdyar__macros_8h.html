Detailed Description

Convenience macros for simplified dynamic array usage.

This file provides quality-of-life macros that simplify the creation and destruction of dynamic arrays with sensible defaults.

Creates a new dynamic array on the heap with default settings.

This macro allocates a dynamic array on the heap, which is useful when:

  • The array needs to outlive the current scope
  • The array is very large
  • You want to pass ownership of the array to another function

Remember to free both the array contents (with DELETE_CDYAR_ONHEAP) and the structure itself when done.

Parameters
namePointer variable name for the array
typeData type of elements (e.g., int, double, struct MyStruct)
capacityInitial capacity of the array
NEW_CDYAR_ONHEAP(my_array, int, 100);
// my_array is a pointer to cdyar_darray
DELETE_CDYAR_ONHEAP(my_array);
