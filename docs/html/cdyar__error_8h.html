Detailed Description

Error handling and reporting system for cdyar.

This file defines error codes, error checking macros, and functions for error reporting and diagnostics.

Macro to validate error code pointer is not NULL.

This macro checks if the error code pointer is NULL before attempting to use it. If NULL, it prints an error message and aborts the program. Use this at the beginning of functions that accept error code pointers.
