cdyar_returncode cdyar_darr ( cdyar_darray *  arr)

Destroys a dynamic array and frees its memory.

Deallocates the array's internal buffer and resets all fields. After calling this function, the array should not be used unless reinitialized with cdyar_narr().

Parameters
arrPointer to the dynamic array to destroy
Returns
CDYAR_SUCCESSFUL on success, error code otherwise

Sets an element at the specified index.

Copies the value pointed to by valueptr into the array at the given index. If CDYAR_ARR_AUTO_RESIZE flag is set and index is out of bounds, the array will automatically resize to accommodate the new element.

Parameters
arrPointer to the dynamic array
indexIndex where the element should be stored
valueptrPointer to the value to copy into the array
Returns
CDYAR_SUCCESSFUL on success, CDYAR_ARR_OUT_OF_BOUNDS if index is invalid and auto-resize is disabled, or other error code
